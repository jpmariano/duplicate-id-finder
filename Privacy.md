Privacy Policy for Duplicate ID Visual Finder

Last Updated: July 13, 2026

Duplicate ID Visual Finder ("the Extension") is a Chrome browser extension designed to help web developers, testers, and other users identify duplicate HTML ID attributes on the currently active webpage.

This Privacy Policy explains how the Extension accesses, uses, stores, and shares information.

1. Information Accessed by the Extension

When a user manually activates Duplicate ID Visual Finder, the Extension temporarily accesses the content of the currently active webpage.

The Extension examines HTML elements and their ID attributes for the sole purpose of identifying duplicate HTML ID values and visually highlighting affected elements.

This webpage analysis occurs locally within the user's browser.

2. Information We Collect

Duplicate ID Visual Finder does not collect personal information.

The Extension does not collect, store, or transmit:

- Names or contact information
- Email addresses
- Authentication information
- Financial or payment information
- Health information
- Personal communications
- Browsing history
- User activity
- Website content
- Location information
- IP addresses
- Analytics or tracking information

The Extension temporarily processes HTML ID attributes from the active webpage to provide its duplicate ID detection functionality. This information is processed locally and is not transmitted to the developer or any external server.

3. How Information Is Used

The Extension accesses and processes webpage information solely to:

- Scan the currently active webpage for HTML ID attributes.
- Identify duplicate HTML ID values.
- Visually highlight webpage elements that contain duplicate IDs.
- Display duplicate ID information to the user.

Webpage information is not used for advertising, profiling, analytics, tracking, or any purpose unrelated to the Extension's duplicate ID detection functionality.

4. Data Storage

Duplicate ID Visual Finder does not store webpage content or HTML ID information on external servers.

Duplicate ID analysis is performed locally in the user's browser.

The Extension does not maintain a database of websites visited, duplicate IDs found, or user activity.

5. Data Transmission

Duplicate ID Visual Finder does not transmit webpage content, HTML ID attributes, browsing activity, or personal information to the developer or to external servers.

The Extension does not use remote servers to process webpage information.

6. Data Sharing

Duplicate ID Visual Finder does not sell, rent, trade, or share user data with third parties.

No user data or webpage information processed by the Extension is shared with:

- Advertisers
- Analytics providers
- Data brokers
- Marketing companies
- Artificial intelligence services
- Other third-party services

7. Chrome Permissions

Duplicate ID Visual Finder uses the following Chrome permissions:

activeTab

The activeTab permission provides temporary access to the currently active webpage after the user manually activates the Extension. This access is used solely to inspect HTML ID attributes and identify duplicate IDs.

scripting

The scripting permission allows the Extension to execute its duplicate ID detection and visual highlighting functionality on the currently active webpage.

These permissions are used only to provide the Extension's stated functionality.

8. Remote Code

Duplicate ID Visual Finder does not use remote code.

All JavaScript and other executable code required by the Extension is included within the Extension package. The Extension does not download or execute JavaScript or other executable code from external sources.

9. Children's Privacy

Duplicate ID Visual Finder does not knowingly collect personal information from children or any other users.

Because the Extension does not collect or transmit personal information, no personal information about children is stored by the developer.

10. Security

Duplicate ID Visual Finder is designed to minimize access to user information.

Webpage analysis is performed locally in the user's browser, and the Extension does not transmit analyzed webpage content to external servers.

11. Chrome Web Store User Data Policy

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

12. Changes to This Privacy Policy

This Privacy Policy may be updated if the functionality or data practices of Duplicate ID Visual Finder change.

Any updates will be reflected on this page with a revised "Last Updated" date.
